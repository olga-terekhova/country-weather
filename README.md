# Analyze visually the weather in selected cities in selected months

This project downloads several months of historical weather forecasts for all major Armenian cities, shown on a map.  
For every city, daily temperature and humidity are extracted and plotted on a chart.  
The user can select the year and the month using a dropdown, and the day using a slider.  
The user can select a city to be highlighted: either using a dropdown, or clicking on the scatter plot, or clicking on the map.
